# Coding Assignment Help - Business Website

## Overview
A modern, responsive landing page website for "Coding Assignment Help" - a company that helps students with programming assignments and projects.

## Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion
- **Backend**: Express.js (minimal - serves static content)
- **Routing**: wouter
- **Fonts**: Poppins, Inter (Google Fonts)
- **Icons**: lucide-react, react-icons (WhatsApp logo)

## Structure
- `client/src/pages/home.tsx` - Main landing page with all sections
- `client/src/App.tsx` - App router
- No database needed - this is a static landing page

## Sections
1. **Hero** - Main headline with code illustration and CTA
2. **Services** - 6 service cards (Java, Python, C/C++, HTML/CSS/JS, MySQL, Projects)
3. **Why Choose Us** - 4 feature blocks
4. **How It Works** - 4-step process
5. **Testimonials** - 3 student testimonials
6. **CTA** - Call to action with gradient background
7. **Footer** - Company info, links, contact

## Key Features
- Sticky navbar with scroll effect
- Smooth scrolling navigation
- Mobile responsive with hamburger menu
- Framer Motion scroll animations
- All "Order Now" buttons link to WhatsApp: https://wa.me/94772095596
- Blue/purple gradient color scheme
